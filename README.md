#### 功能列表

- 首頁可以快速分類專案類型
- 每頁都可以用關鍵字快速搜尋
-   -沒有該關鍵字會跳提示
-   -沒有輸入訊息會跳提示
- 儲存工種時，有防呆不會重複開出工種
